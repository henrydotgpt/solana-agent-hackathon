## 4. SOLANA DEVELOPER TOOLS & SDKS

### Helius (RPC + Data Infrastructure)
- **RPC Nodes:** High-performance Solana RPC with optimization techniques
- **DAS API (Digital Asset Standard):** Get all user assets in one call â€” tokens, NFTs, prices
  - `getAssetsByOwner`, `searchAssets`, `getTokens`
- **Enhanced Transactions API:** Decoded transaction history (SWAP, NFT_SALE, TRANSFER types)
  - `getTransactionsForAddress` â€” now includes associated token account txs
- **Helius Sender:** Ultra-low latency tx submission (staked connections + Jito dual routing)
- **Data Streaming:** Real-time event listening / webhooks
- **Helius MCP:** AI agent-friendly Model Context Protocol endpoint
- **ZK Compression:** Compressed NFTs, cost-efficient on-chain data
- **Staking API:** Programmatic staking via Helius SDK
- **Orb Explorer:** Token, market, wallet, validator explorer

### Jupiter (DEX Aggregation)
- **Ultra Swap API:** Most advanced swap solution on Solana
  - Juno Liquidity Engine (multi-source, self-learning)
  - Best executed price (predictive execution + slippage-aware routing)
  - Sub-second landing via Jupiter Beam
  - Real-Time Slippage Estimator (RTSE)
  - Gasless support (via JupiterZ RFQ)
- **Limit Orders:** On-chain limit order book
- **DCA:** Dollar-cost averaging
- **Perpetuals:** Perps trading
- **API:** `GET /quote`, `POST /swap`, price feeds

### Solana Agent Kit (SendAI)
- **60+ Solana actions** available as AI tools
- Plugin architecture:
  - `@solana-agent-kit/plugin-token` â€” SPL ops, swaps, bridges
  - `@solana-agent-kit/plugin-nft` â€” Metaplex NFTs
  - `@solana-agent-kit/plugin-defi` â€” Staking, lending, perps
  - `@solana-agent-kit/plugin-misc` â€” Airdrops, price feeds, domains
  - `@solana-agent-kit/plugin-blinks` â€” Solana Blinks/Actions
- Integrations: Jupiter, Raydium, Orca, Meteora, Drift, Jito, Marinade, Pyth, Metaplex, Wormhole
- LangChain + Vercel AI SDK ready
- V2 with modular plugin system

### Anchor Framework
- Solana smart contract framework (Rust)
- PDA-based account architecture
- Used by nearly every hackathon project

### Metaplex
